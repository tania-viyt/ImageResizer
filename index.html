О приложении

ImageResizer - приложение созданное для быстрого изменения размеров изображения. Поддерживает пакетную обработку и обработку в ручном режиме.

Системные требования

  • Microsoft Windows 7 / 8 / 8.1 / 10 (32- или 64-битные) или macOS Version 10.12 и выше.
  • Память: ОЗУ 512 Mb.
  • Жесткий диск: не менее 100 МБ свободного дискового пространства.
  • Разрешение: 1200 800 и выше.
  • Java runtime environment: Version 8 Update 131 и выше.

Поддерживаемые форматы

ImageResizer поддерживает следующие форматы: jpg, png, bmp.
