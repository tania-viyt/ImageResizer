О приложении

ImageResizer - приложение созданное для изменения размеров изображений в несколько кликов. Поддерживает пакетную обработку и обработку в ручном режиме.

Системные требования для Windows

  • Microsoft Windows XP / 7 / 8 / 8.1 / 10 (32- или 64-битные).
  • RAM: 128 МБ
  • Жесткий диск: не менее 134 МБ свободного дискового пространства.
  • Java runtime environment: Version 8 Update 131 и выше (Разрядность в зависимости от разрядности системы).

Системные требования для macOS

  • Компьютер Mac на базе процессора Intel под управлением ОС Mac версии 10.7.3 и выше
  • Java runtime environment: Version 8 Update 131 и выше

Поддерживаемые форматы

ImageResizer поддерживает следующие форматы:

  • jpg
  • png
  • bmp
